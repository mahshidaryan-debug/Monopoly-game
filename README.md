# Monopoly Game (Python + Tkinter)

## Overview

This project is a simple Monopoly-style board game implemented using **Python** and **Tkinter**.
The goal of this project was to practice **GUI development**, **event-driven programming**, and **basic game logic implementation**.

The game includes player movement, dice rolling mechanics, and simple interactions with the game board.

---

## Features

* Graphical user interface built with Tkinter
* Dice rolling system
* Player movement across the board
* Turn-based gameplay logic
* Basic board interaction

---

## Technologies Used

* Python
* Tkinter

---

## How to Run

1. Make sure Python is installed on your system.
2. Clone the repository:

```
git clone https://github.com/mahshidaryan-debug/monopoly-game.git
```

3. Navigate to the project folder:

```
cd monopoly-game
```

4. Run the game:

```
python main.py
```
